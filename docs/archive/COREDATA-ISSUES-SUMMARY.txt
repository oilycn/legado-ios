🚨 【CRITICAL】必须立即修复的问题 
=============================================================================

1. 主键设计不一致
   ├─ iOS: bookId (UUID) - 生成的唯一标识
   ├─ Android: bookUrl (String PrimaryKey) - 网络标识  
   └─ 影响: 数据导入/导出会失败，无法跨平台同步

2. 缺少用户定制字段 (3 个)
   ├─ customCoverUrl: String? (自定义封面 URL)
   ├─ customIntro: String? (自定义简介)
   └─ 影响: 用户修改的内容无法保存

3. 缺少书籍类型字段
   ├─ type: Int32 (枚举: text=0, audio=1, image=2)
   └─ 影响: 无法区分书籍类型，阅读器选择错误

4. 完全缺少阅读配置对象 ReadConfig
   ├─ 字段: reverseToc, pageAnim, imageStyle, useReplaceRule, ttsEngine 等
   ├─ 方法: 对应的 16+ 个 getter/setter
   └─ 影响: 用户阅读设置无法保存 (翻页方式、字体、主题等)

5. 规则存储方式不当
   ├─ iOS: ruleSearchData 等存为 Data? (二进制 JSON)
   ├─ 缺失: ExploreRule, ReviewRule 的 Codable 结构体
   └─ 影响: 规则无类型检查，易出现运行时错误
